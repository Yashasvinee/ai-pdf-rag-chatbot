## DOCUMIND AI-AI PDF RAG Chatbot

An AI-powered chatbot that lets users upload PDF documents and ask questions about their contents using Retrieval-Augmented Generation (RAG).

## Features

- Upload PDF documents
- Extract and split text into chunks
- Generate embeddings using Google Gemini
- Store embeddings with FAISS
- Retrieve relevant context using semantic search
- Generate answers with Gemini
- ChatGPT-style interface using Streamlit

## Tech Stack

- Python
- Streamlit
- LangChain
- Google Gemini API
- FAISS
- PyPDF
- Python Dotenv
